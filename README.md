# azcam-monitor

azcam-monitor is a monitoring app for azzcam processes running on a computer.

## Documentation

See https://azcam.readthedocs.io.

## Installation Example

```shell
git clone https://github.com/mplesser/azcam
git clone https://github.com/mplesser/azcam-monitor
pip install -e azcam
pip install -e azcam-monitor
```
